我的第一篇博客文章

Hexo搭建个人博客

搭建过程

1.安装nodejs
2.因为npm在国外，所以 安装淘宝的源
npm install -g cnpm –registry=https:// registry.npm.taobao.org


3.安装好之后可以使用cnpm -v 查看版本

4.安装Hexo框架
cnpm install -g hexo-cli

5.使用hexo-v验证一下

6.新建一个文件夹 如blog
在doc窗口中进入该文件夹，然后windows下使用命令
hexo init

然后 （如果报错就是用命令 cnpm install ）使用
hexo S 启动项目就可以在浏览器看到默认的页面了
