Hexo搭建个人博客<二>

1.主题的更换

Hexo默认带了一个主题，默认的主题是themes文件夹中的landscape主题，一般需要更改主题，这里介绍目前比较火的nexT主题。

1.1 主题的下载

在存放博客的根目录，Git Bash here ,然后输入以下命令
git clone https://github.com/theme-next/hexo-theme-next themes/next
然后他就会自己去下载主题，等待下载好后，就可以在themes文件夹中看到多一个next的文件夹，说明已经下载好主题。

    2.新博客的创建

    2.1 新博客的创建
    主题更改后就可以新建一个博客了，我们使用命令
    hexo n "first-blog"
    就可以创建一个新的文件，在)中新建一个first-blog.md文件，我们可以使用各种工具进行博客的编写，推荐大家使用MarkDown的语法进行编写这里有MarkDown语法的教程，本文不介绍博客的具体书写方法，写完成后，就可以三连然后启动查看效果了。

    2.2 设置文章为预览模式

    如果文章不设置为预览模式，那么整个文章都会展开，在后面文章变多后就显的很乱，所以设置预览模式。
    在next的配置文件中，搜索auto_excerpt，如图修改)这样在首页的文章都只会显示预览的文字。这样在写文章的正文之前使用如图的方式添加预览的概况。，我们也可以如图给添加标签和分类，<!--more-->标签之前的文字只会在预览的时候看到。
