完整版如何使用Hexo以及GitHub.io来搭建自己的博客

Hexo搭建个人博客

tips: 作者在搭建之前是已经安装git的，方便后面的命令行操作

1.安装Node.js

  • 直接在官网下载对应的操作系统的软件，然后直接下一步下一步就可以安装好，软件默认就会配环境变量的。
    安装完成后，cmd(或者直接在任意目录Git Bash Here), 输入 node -v 能出现对应的版本号，输入 npm -v 也能出现对应的版本号，说明安装成功，如图所示

    2.安装 cnpm

  • 2.1 因为npm在国外，所以我们安装淘宝的源 安装如图
  • 2.2 安装好之后可以使用cnpm -v 查看版本

3.安装Hexo框架

  • 3.1 安装Hexo 很简单，就一个命令
  • 3.2 使用hexo-v验证一下

    至此，我们的环境就准备好了。

4. 创建博客

  • 新建一个文件夹，用来存放你博客的东西，这里我就随意建一个叫blog的文件夹，进入文件夹，然后使用Git Bash Here 进入该文件夹。
  • 使用hexo init 命令来初始化这个博客文件夹，在初始化后，hexo会自动到github下载一些主题文件之类的东西
  • 如果初始化过程中报错乱码,那个就是用这个 cnpm install命令，输入几次，直到初始化成功
  • 初始化完成后，接下来就是hexo三连了。
    一次输入三个命令 三个命令分别是清理缓存，生成代码，启动博客。

正常情况下，在最后一个命令后，控制台就会出现项目在4000端口启动，如图

此时，打开浏览器访问 localhost:4000就会看到博客了，默认会有一个Hello World! 的博客。

总结

到这里我们基本的博客就可以搭建起来了，接下来的教程还会继续分享如何新建博客、更改主题、美化页面等内容。
